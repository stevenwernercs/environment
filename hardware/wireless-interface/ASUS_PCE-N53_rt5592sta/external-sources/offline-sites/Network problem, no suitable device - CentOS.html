      Postby giulix63 » 2016/03/29 08:35:12

      Why are you trying to start the network directly? It looks like NetworkManager is enabled on that system (NM_CONTROLLED=yes). It's its job to start the network. Reboot the system and post the output of the following commands run as root, please:

      Code: Select all

      systemctl -l status NetworkManager
      ip addr show
      ip route show
      cat /etc/resolv.conf

      Then, if NetworkManager is up&running, run

      Code: Select all

      journalctl -b -lu NetworkManager >nm.log

      as root, upload nm.log here and post the link, please.
      Root is evil: Do not use root (sudo) to run any of the commands specified in my posts unless explicitly indicated. Please, provide the necessary amount of context to understand your problem/question.
